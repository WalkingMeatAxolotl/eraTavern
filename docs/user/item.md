# 物品与服装

## 概述

**物品**和**服装**是游戏中角色可以持有和使用的实体。

- **物品**：存放在角色背包中的对象，如食物、药水、钥匙等。物品可以堆叠，可以在行动中被消耗或获得。
- **服装**：角色穿戴的装备，占据特定的身体槽位。服装可以提供属性加成，并且外层服装会遮挡内层服装。

两者都在扩展包中定义，通过物品编辑页和服装编辑页分别管理。

---

## 物品定义

每个物品由以下字段定义：

| 字段 | 类型 | 说明 |
|------|------|------|
| **id** | 字符串 | 唯一标识符，加载后自动加上扩展包命名空间（如 `base.apple`） |
| **name** | 字符串 | 显示名称（如「苹果」） |
| **tags** | 字符串数组 | 标签列表，用于分类和筛选（如 `["食物", "水果"]`） |
| **description** | 字符串 | 物品描述文本 |
| **maxStack** | 数字 | 最大堆叠数量。设为 `1` 表示不可堆叠 |
| **sellable** | 布尔值 | 是否可以出售 |
| **price** | 数字 | 出售价格（仅在 sellable 为 true 时有意义） |

**示例**：一个面包物品可能是 `id: bread`、`name: 面包`、`tags: ["食物"]`、`maxStack: 10`。

---

## 标签系统

标签用于对物品进行分类。标签系统维护一个**标签池**，所有物品共享同一组可用标签。

### 管理标签池

- 在物品编辑页顶部可以管理标签池
- **添加标签**：输入新标签名并添加到池中
- **删除标签**：从池中移除不再需要的标签

### 为物品分配标签

- 编辑物品时，从标签池中选择要分配的标签
- 一个物品可以拥有多个标签
- 同一个标签可以被多个物品共享

标签的主要用途是方便在物品管理器中按类别浏览和筛选物品。

---

## 物品管理器

物品管理器提供两种浏览视图，方便你从不同角度管理物品：

### 按标签视图（byTag）

以标签为主维度展示：

```
食物
  ├── 苹果
  ├── 面包
  └── 烤鱼
武器
  ├── 短剑
  └── 长弓
```

每个标签下列出拥有该标签的所有物品。点击物品可以跳转到编辑。

### 按物品视图（byItem）

以物品为主维度展示：

```
苹果    [食物] [水果]
面包    [食物]
短剑    [武器]
```

每个物品旁边显示它的所有标签。

### 交叉引用提示（Tooltip）

在两种视图中，悬停在物品或标签上时会弹出提示框，显示交叉引用信息：
- 在按标签视图中悬停物品 → 显示该物品的所有标签
- 在按物品视图中悬停标签 → 显示该标签下的所有物品

## 物品库存

物品在角色身上以**库存**的形式存在。每个库存条目包含：

| 字段 | 说明 |
|------|------|
| **物品 ID** | 引用物品定义 |
| **数量（amount）** | 当前持有数量 |

当数量超过 1 时，物品会自动堆叠显示。堆叠上限由物品定义中的 `maxStack` 决定——超过上限的部分会占据新的库存格。

角色的库存可以在角色编辑页中直接编辑，也可以通过行动效果在游戏过程中动态增减。

---

## 服装定义

每件服装由以下字段定义：

| 字段 | 类型 | 说明 |
|------|------|------|
| **id** | 字符串 | 唯一标识符 |
| **name** | 字符串 | 显示名称（如「铁胸甲」） |
| **slot** | 字符串 | 占据的装备槽位（见下表） |
| **occlusion** | 字符串数组 | 穿着时遮挡的槽位列表 |
| **effects** | 效果数组 | 穿着时对角色属性的加成效果 |

---

## 装备槽位

角色身体分为以下装备槽位：

| 槽位 ID | 说明 | 示例 |
|---------|------|------|
| `hat` | 头部 | 帽子、头盔、发饰 |
| `upperBody` | 上身外衣 | 外套、胸甲、上衣 |
| `upperUnderwear` | 上身内衣 | 背心、内衫 |
| `lowerBody` | 下身外衣 | 裤子、裙子、腿甲 |
| `lowerUnderwear` | 下身内衣 | 内裤 |
| `hands` | 手部 | 手套、护腕 |
| `feet` | 足部（袜类） | 袜子、裹脚布 |
| `shoes` | 鞋子 | 靴子、凉鞋 |
| `accessory1` | 饰品栏 1 | 项链、护身符 |
| `accessory2` | 饰品栏 2 | 戒指、耳环 |
| `accessory3` | 饰品栏 3 | 手镯、徽章 |

每个槽位同时只能装备一件服装。

---

## 遮挡系统

遮挡（occlusion）是服装系统的核心机制：**外层服装穿着时，会遮挡住内层服装的可见性。**

### 工作原理

每件服装可以定义一个 `occlusion` 列表，指定它会遮挡哪些槽位。

**示例**：一件外套的定义可能是：
- `slot`: `upperBody`（占据上身外衣槽位）
- `occlusion`: `["upperUnderwear"]`（遮挡上身内衣）

当角色穿着这件外套时，内衣虽然仍然装备着，但会被标记为「被遮挡」——在游戏描述中不可见。

### 遮挡规则

- 只有 **worn**（完整穿着）状态的服装才会产生遮挡
- **halfWorn**（半穿）状态不产生遮挡——比如外套敞开时，内衣可见
- 遮挡是单向的：外层遮挡内层，内层不影响外层

---

## 穿着状态

每件装备的服装有三种穿着状态：

| 状态 | 说明 | 是否产生遮挡 |
|------|------|:------------:|
| **worn** | 完整穿着 | 是 |
| **halfWorn** | 半穿状态（部分脱下、敞开等） | 否 |
| **none** | 未装备（槽位为空） | — |

**halfWorn 的用途**：表达服装被部分脱下或松开的状态。例如外套从 worn 变为 halfWorn 后，它不再遮挡内层服装，但仍然占据着槽位。这在叙事场景中非常有用。

穿着状态可以通过行动效果在游戏中动态改变。

---

## 服装效果

服装可以附带效果（effects），在穿着时对角色属性产生影响。效果的格式与特质效果相同：

| 字段 | 说明 |
|------|------|
| **目标属性** | 要影响的角色资源或能力值 |
| **方向** | increase（增加）或 decrease（减少） |
| **类型** | fixed（固定值）或 percentage（百分比） |
| **数值** | 具体的加成数值 |

**示例**：

| 服装 | 效果说明 |
|------|---------|
| 铁胸甲 | 防御力 increase fixed 5 |
| 魔法戒指 | 魔力 increase percentage 10 |
| 沉重铠甲 | 敏捷 decrease fixed 3 |

效果仅在服装被穿着（worn 或 halfWorn）时生效，脱下后自动移除。

## 在行动中使用

行动可以通过以下方式与物品和服装交互：

### 物品消耗（Item Costs）

在行动的消耗条件中指定需要消耗的物品：

| 设置 | 说明 |
|------|------|
| 物品 ID | 要消耗的物品 |
| 数量 | 消耗的数量 |

执行行动时会自动扣除指定数量的物品。如果物品不足，行动不可执行。

### 物品效果（Item Effects）

在行动结果中添加或移除物品：

| 操作 | 说明 |
|------|------|
| 添加物品 | 向角色背包中增加指定数量的物品 |
| 移除物品 | 从角色背包中移除指定数量的物品 |

### 服装效果（Clothing Effects）

在行动结果中改变服装的穿着状态：

| 操作 | 说明 |
|------|------|
| 装备服装 | 将指定服装装备到角色身上（状态变为 worn） |
| 改变穿着状态 | 将已装备服装的状态在 worn / halfWorn / none 之间切换 |

通过组合这些效果，可以实现丰富的游戏逻辑。

## 典型用例

### 创建一组食物物品

1. 在物品编辑页的标签池中添加标签：`食物`、`饮品`
2. 创建物品「面包」：tags 设为 `["食物"]`，maxStack 设为 `10`
3. 创建物品「苹果」：tags 设为 `["食物"]`，maxStack 设为 `20`
4. 创建物品「啤酒」：tags 设为 `["饮品"]`，maxStack 设为 `5`，sellable 设为 true，price 设为 `3`
5. 在物品管理器中用「按标签」视图检查分类是否正确
6. 创建一个「吃东西」行动，消耗 1 个食物类物品，恢复角色的饱食度资源

### 创建有遮挡关系的服装

1. 创建服装「丝质内衫」：slot 设为 `upperUnderwear`，无遮挡
2. 创建服装「皮革外套」：slot 设为 `upperBody`，occlusion 设为 `["upperUnderwear"]`
3. 创建服装「棉质内裤」：slot 设为 `lowerUnderwear`，无遮挡
4. 创建服装「长裤」：slot 设为 `lowerBody`，occlusion 设为 `["lowerUnderwear"]`

穿着效果：
- 当「皮革外套」为 **worn** 状态时，「丝质内衫」被遮挡不可见
- 当「皮革外套」变为 **halfWorn** 状态时，「丝质内衫」恢复可见
- 创建一个「敞开外套」行动，将「皮革外套」的状态从 worn 改为 halfWorn
