# 特质与属性

## 概述

特质（Trait）定义了角色的各种属性标签。系统中有两大类特质：

| 类型 | 说明 | 示例 |
|------|------|------|
| **普通特质** | 布尔型，角色拥有或不拥有 | 人类、勇敢、识字 |
| **能力（Ability）** | 带经验值的数值型属性，有等级评定 | 剑术、魔法、烹饪 |

此外，还有一种特殊类型**经验（Experience）**，用于自动追踪角色经历过的事件次数。

特质可以附带**效果**，影响角色的资源、能力或基础信息。多个相关特质可以通过**特质组**进行分组管理。基于角色属性还可以定义**衍生变量**，用公式动态计算数值。

## 普通特质

普通特质是最基础的特质类型——角色要么拥有它，要么没有。每个特质属于一个**分类（category）**：

| 分类 | 说明 | 示例 |
|------|------|------|
| `race` | 种族 | 人类、精灵、矮人 |
| `bodyTrait` | 身体特征 | 强壮、灵活、虚弱 |
| `mentalTrait` | 心理特征 | 勇敢、谨慎、冲动 |
| `techTrait` | 技术/知识特征 | 识字、炼金术、骑术 |
| `other` | 其他 | 贵族血统、被诅咒 |

分类主要用于编辑器中的筛选和特质组的过滤，不影响游戏逻辑。

**创建普通特质时**，填写 ID、名称、描述，选择分类即可。

## 能力（Ability）

能力是 `category="ability"` 的特质，具有经验值系统。

### 经验值与等级

每个能力有一个 0 到 8000+ 的**经验值（experience points）**，系统根据经验值自动评定等级：

| 等级 | 经验值范围 | 说明 |
|------|-----------|------|
| **G** | 0 – 999 | 入门 |
| **F** | 1000 – 1999 | 初学 |
| **E** | 2000 – 2999 | 基础 |
| **D** | 3000 – 3999 | 熟练 |
| **C** | 4000 – 4999 | 精通 |
| **B** | 5000 – 5999 | 卓越 |
| **A** | 6000 – 6999 | 大师 |
| **S** | 7000+ | 传奇 |

### 默认值（defaultValue）

创建能力时可以设置**默认值**。当角色拥有该能力但未特别指定经验值时，将使用此默认值作为初始经验。

### 与普通特质的区别

- 普通特质：角色拥有或不拥有（布尔）
- 能力：角色拥有后，还有一个具体的经验值数值，可以增减、衰减

## 能力衰减

能力可以配置**衰减（decay）**，使经验值随时间自动下降，模拟"不用则废"的效果。

衰减使用累积制计算：游戏时间每推进一个 tick（5 分钟），系统会累积经过的时间。当累积时间达到设定的衰减间隔时，触发一次衰减。衰减对所有角色（包括 NPC）同时生效。

衰减配置包含三个参数：

| 参数 | 说明 |
|------|------|
| `amount` | 每次衰减的数值 |
| `type` | 衰减方式：`fixed`（固定值）或 `percentage`（百分比） |
| `intervalMinutes` | 衰减间隔（游戏时间，单位：分钟） |

**示例**：

- `amount=10, type=fixed, intervalMinutes=1440`：每过 1 天（1440 分钟），经验值减少 10 点
- `amount=5, type=percentage, intervalMinutes=720`：每过 12 小时，经验值减少当前值的 5%

## 经验（Experience）

经验是 `category="experience"` 的特殊特质，具有以下特点：

- **自动应用**：经验特质会自动添加到所有角色身上，不需要手动分配
- **事件追踪**：记录角色经历某类事件的**次数**和**首次发生时间**
- 适合用于追踪角色的经历，如"被击败次数"、"完成交易次数"等

经验特质通常由行动结果自动更新，而非手动编辑。

## 特质效果

特质可以附带**效果（effects）**，在角色拥有该特质时自动生效。效果可以影响：

| 影响目标 | 说明 | 示例 |
|----------|------|------|
| **资源（resource）** | 角色的资源数值 | 生命值 +20 |
| **能力（ability）** | 其他能力的经验值 | 剑术经验 +500 |
| **基础信息（basicInfo）** | 角色的基础属性 | 移动速度变化 |

### 效果类型

| 类型 | 说明 | 示例 |
|------|------|------|
| **increase** | 增加数值 | 生命上限 +50 |
| **decrease** | 减少数值 | 移动速度 -10 |

### 固定值与百分比

- **固定值**：直接加减指定数值（如 +50）
- **百分比**：填写的数值代表百分比形式。例如填写 `120` 增加 = +20%，填写 `80` 减少 = -20%
- **叠加规则**：多个百分比效果**相加叠加**。两个 +20% = 总计 +40%，而非连乘

**计算顺序**：先加固定值，再乘百分比。`最终值 = (基础值 + 固定值总和) × (1 + 百分比增量总和)`

**示例**：一个"虚弱"特质效果：生命上限 减少 百分比 80（-20%，即生命上限降至 80%）。

## 特质组

特质组（Trait Group）将多个相关特质归为一组，便于管理和分配。

### 排他性

| 模式 | 说明 | 适用场景 |
|------|------|----------|
| **互斥（exclusive）** | 组内只能选择一个特质 | 种族选择——一个角色只能是一个种族 |
| **非互斥（non-exclusive）** | 组内可以选择多个特质 | 性格特征——一个角色可以既勇敢又冲动 |

### 分类过滤

创建特质组时可以指定一个**分类**过滤器。设置后，该组只会显示指定分类下的特质供选择。例如，种族组设置过滤分类为 `race`，则只列出种族类特质。

## 衍生变量

衍生变量（Variable / VariableDefinition）是通过公式动态计算的数值，可以引用角色的各种属性。衍生变量可以用在行动的条件判断和效果中。

### 步骤公式

每个变量由一系列**步骤（step）** 组成，从初始值开始，逐步运算得出最终结果。

### 步骤类型

每个步骤先获取一个值，然后与当前结果进行运算。可用的取值类型：

| 步骤类型 | 说明 | 示例 |
|----------|------|------|
| `constant` | 固定常数 | 100 |
| `ability` | 角色某个能力的经验值 | 剑术的经验值 |
| `resource` | 角色某个资源的当前值 | 当前生命值 |
| `basicInfo` | 角色基础信息的某个字段 | 年龄 |
| `traitCount` | 角色拥有的某分类特质数量 | 拥有几个 bodyTrait |
| `hasTrait` | 角色是否拥有某特质（有=1，无=0） | 是否拥有"强壮" |
| `experience` | 角色某个经历的触发次数 | 被夸奖过几次 |
| `itemCount` | 角色背包中某物品的数量 | 持有几瓶药水 |
| `variable` | 引用另一个衍生变量的值 | 引用"基础攻击力" |

### 运算操作

每个步骤可选的运算操作：

| 操作 | 说明 |
|------|------|
| `add` | 加（当前值 + 步骤值） |
| `subtract` | 减（当前值 - 步骤值） |
| `multiply` | 乘（当前值 × 步骤值） |
| `divide` | 除（当前值 ÷ 步骤值） |
| `min` | 取较小值 min(当前值, 步骤值) |
| `max` | 取较大值 max(当前值, 步骤值) |
| `floor` | 下限钳制 max(当前值, 步骤值)，确保不低于步骤值 |
| `cap` | 上限钳制 min(当前值, 步骤值)，确保不超过步骤值 |

**示例**：定义一个"综合战力"变量：

```
步骤1: constant 0          → 初始值 = 0
步骤2: ability "剑术" add   → 0 + 剑术经验
步骤3: ability "魔法" add   → + 魔法经验
步骤4: constant 2 divide   → ÷ 2（取平均）
步骤5: constant 0 max      → 确保不小于 0
```

## 变量测试

编辑器提供**测试面板**，可以即时验证变量公式是否正确：

1. 在变量编辑页面找到测试区域
2. **选择角色**：从下拉列表中选择一个角色
3. **点击评估**：系统会基于该角色的实际属性计算变量值
4. **查看逐步追踪**：结果会显示每一步的取值和运算过程，方便调试

通过测试面板可以快速验证公式是否按预期工作，及时发现引用错误或计算偏差。

## 典型用例

### 创建种族特质组

1. **创建特质**：在特质编辑页新建若干特质，分类选 `race`
   - 人类（ID: `human`）
   - 精灵（ID: `elf`）— 添加效果：生命上限 decrease 百分比 90
   - 矮人（ID: `dwarf`）— 添加效果：生命上限 increase 固定值 20
2. **创建特质组**：新建特质组"种族"
   - 设为**互斥**模式
   - 分类过滤设为 `race`
3. **分配给角色**：在角色编辑页，从种族组中为角色选择一个种族

### 创建带衰减的能力

1. **创建能力特质**：新建特质，分类选 `ability`
   - ID: `swordsmanship`，名称: 剑术
   - 默认值: `1000`（F 级起步）
   - 衰减配置: `amount=5, type=fixed, intervalMinutes=1440`（每天衰减 5 点）
2. **分配给角色**：在角色编辑页添加该能力，可以调整初始经验值
3. 角色如果长时间不进行与剑术相关的行动，经验会逐渐降低

### 创建衍生战力变量

1. **创建变量**：在变量编辑页新建变量
   - ID: `combatPower`，名称: 综合战力
2. **编辑公式步骤**：
   - 步骤 1：`constant 0`（初始值）
   - 步骤 2：`ability "swordsmanship" add`（加剑术经验）
   - 步骤 3：`ability "magic" add`（加魔法经验）
   - 步骤 4：`resource "hp" add`（加当前生命值）
   - 步骤 5：`constant 3 divide`（除以 3 取平均）
3. **测试**：选择一个角色，点击评估，检查每步计算是否正确
4. **使用**：在行动条件中引用该变量（如"综合战力 > 2000 才能挑战 Boss"）
