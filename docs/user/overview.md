# 总览

## 这是什么

AI Tavern Game 是一个文字冒险游戏引擎。你可以通过配置 JSON 数据来创建自己的游戏世界——定义角色、地图、行动、物品等，然后在其中进行游戏。

## 核心概念

### 世界（World）

**世界是你的游戏存档空间。** 每个世界是一个独立的游戏环境，包含：
- 启用了哪些扩展包
- 谁是玩家角色
- 游戏存档

你可以创建多个世界，每个世界可以启用不同的扩展包组合。世界之间互不影响。

**操作入口**：点击顶部导航栏左侧的 **[世界]** 按钮打开世界侧边栏。

### 扩展包（Addon）

**扩展包是游戏内容的来源。** 所有游戏实体——角色、地图、特质、行动、物品、服装、事件——都定义在扩展包中。

一个扩展包通常围绕一个主题，比如：
- 「基础包」：通用奇幻设定的种族、特质、基础行动
- 「红魔馆」：特定场景的角色、地图、专属行动

扩展包可以：
- **独立使用**：自包含，不依赖其他包
- **组合叠加**：多个包同时启用，后启用的可以扩展或覆盖先启用的

**操作入口**：点击顶部导航栏右侧的 **[扩展]** 按钮打开扩展侧边栏。

### 版本与分支

每个扩展包可以有多个**版本**。版本分两种：

| 类型 | 说明 | 示例 |
|------|------|------|
| **本体** | 原始版本，可被多个世界共享 | `1.0.0` |
| **分支** | 某个世界的专属副本 | `1.0.0-myworld` |

**为什么需要分支？** 当你在世界 A 中修改了某个扩展包的内容（比如调整了某个角色的属性），你不希望这个修改影响到世界 B。分支机制会自动为每个世界创建独立的副本。

- 首次在世界中启用扩展包时，系统会询问你是**创建专属分支**还是**直接使用本体**
- 选择分支：修改只影响当前世界
- 选择本体：修改会影响所有使用该本体的世界

### 保存

游戏中的编辑操作（修改角色属性、编辑地图、调整行动定义等）会暂存在内存中。你需要点击底部的 **[保存变更]** 按钮才会写入磁盘。

此外，你还可以在设置页面创建**存档槽位**，保存和加载游戏进度。

## 界面布局

```
┌─────────────────────────────────────────────────────────┐
│  [世界] ← 世界名称  │  游戏  地图  角色  ...  │ [扩展] →  │  ← 导航栏
├────────┬────────────────────────────────┬────────────────┤
│        │                                │                │
│  世界  │         主内容区域              │    扩展包      │
│ 侧边栏 │    （游戏界面 / 编辑器）         │    侧边栏     │
│        │                                │                │
│（左侧） │                                │  （右侧）      │
│        │                                │                │
├────────┴────────────────────────────────┴────────────────┤
│                    [保存变更]                              │  ← 浮动操作栏
└─────────────────────────────────────────────────────────┘
```

- **导航栏**：切换页面（游戏、地图编辑、角色编辑等），左右两端是侧边栏开关
- **世界侧边栏**（左）：管理世界——创建、切换、删除、编辑信息
- **扩展侧边栏**（右）：管理扩展包——启用/禁用、版本管理、编辑元数据
- **主内容区域**：根据导航栏选择显示不同的编辑器或游戏界面
- **浮动操作栏**：有未保存修改时出现，提供 [保存变更] 和 [撤销变更]

## 游戏实体一览

以下是你可以在扩展包中定义的所有实体类型：

| 实体 | 说明 | 编辑入口 |
|------|------|----------|
| **角色** | 游戏中的人物，包含基础信息、资源、特质、能力、装备、物品、位置 | 角色编辑页 |
| **地图** | 游戏世界的空间，由格子组成，格子之间有连接关系 | 地图编辑页 |
| **特质** | 角色的属性标签，如种族、性格等。分为普通特质和能力（有经验值） | 特质编辑页 |
| **特质组** | 将相关特质分组，可设为互斥（只能选一个）或多选 | 特质组编辑页 |
| **物品** | 可持有的物品，有标签、堆叠上限、价格等属性 | 物品编辑页 |
| **服装** | 可穿戴的装备，占据特定槽位，可遮挡其他槽位 | 服装编辑页 |
| **行动** | 角色可执行的操作，包含条件、消耗、结果等。NPC 也会根据权重选择行动 | 行动编辑页 |
| **衍生变量** | 基于角色属性计算的动态数值，可用于行动条件和效果 | 变量编辑页 |
| **全局事件** | 满足条件时自动触发的效果，如天气变化、定时事件 | 事件编辑页 |
| **世界变量** | 全局数值/开关，用于追踪世界状态（如剧情进度） | 世界变量编辑页 |

## 典型使用流程

### 创建新游戏

1. 打开世界侧边栏 → 点击 **+** 创建新世界
2. 输入世界 ID 和名称
3. 打开扩展侧边栏 → 启用需要的扩展包
4. 弹窗选择：创建世界专属分支（推荐）或直接使用本体
5. 点击 [保存变更]
6. 在角色编辑页选择玩家角色

### 编辑游戏内容

1. 通过导航栏进入对应编辑器（地图、角色、行动等）
2. 编辑内容（所有修改暂存在内存中）
3. 编辑完成后点击底部 [保存变更] 写入磁盘

### 进行游戏

1. 切换到「游戏」页面
2. 查看当前场景描述、可用行动
3. 选择行动执行
4. 观察结果，NPC 也会自动执行行动

### 管理存档

1. 进入设置页面
2. 创建存档槽位保存当前进度
3. 随时加载之前的存档
